Programa na linguagem de programação java em ambiente janelas (GUI), com as seguintes funcionalidades:

 - Permitir fazer medição sonora através de um sonómetro (guardando as medições em áudio e em texto) - construir de forma a que possa suportar diversos tipos de sonómetros.

 - Possibilidade de "encriptação" do áudio para não se perceber o que as pessoas dizem mas, mesmo assim, mantendo inalterado o valor global dos níveis sonoros -> podem ser feitos alguns testes com métodos já existentes ou tentar desenvolver um método novo.

 - Mostragem dos dados em tabelas/gráficos interactivos.

 - Análise dos dados através de FFT (Fast Fourier Transform) e outros métodos matemáticos de análise - de acordo com indicações de normas europeias de ruído.

 - Possibilidade da importação de dados a partir de ficheiros gravados noutra altura e fazer também aqui a análise.

Por fim, se tudo tiver sido feito sem grandes percalços, posso começar a estruturar a parte de audio event detection (reconhecimento de padrões dentro do áudio).
